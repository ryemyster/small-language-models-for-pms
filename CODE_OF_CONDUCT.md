# Code of Conduct

## The short version

Be direct. Be honest. Be kind. Don't be a jerk.

---

## The longer version

This project is a learning resource. People contributing to it — and people asking questions about it — may have no prior experience with code, machine learning, or terminals. That's the point.

**Expected behavior:**

- Treat contributors and users with patience and respect
- Give feedback that is specific and actionable, not dismissive
- Acknowledge when something is confusing — that's usually a documentation problem, not a user problem
- Disagree with ideas, not people

**Not acceptable:**

- Condescension toward beginners or non-technical people
- Dismissing questions as "too basic"
- Personal attacks or harassment of any kind
- Language that excludes or demeans

---

## Enforcement

If you see behavior that violates this — open an issue or contact the repo maintainer directly. Repeated violations result in removal from the project.

---

This code of conduct applies to all spaces associated with this project: GitHub issues, pull requests, and any other community spaces.
